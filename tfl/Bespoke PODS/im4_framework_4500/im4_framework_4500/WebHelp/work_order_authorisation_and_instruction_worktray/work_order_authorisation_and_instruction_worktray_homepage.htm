Work Order Authorisation and Instruction Worktray Homepage

Figure 105

The Work Order Authorisation and Instruction Worktray Homepage will display a histogram displaying the following work orders awaiting instruction:

t                        Work Orders raised in the last 2 days

t                        Work Orders raised in the last 3-5 days

t                        Work Orders raised in the last 5+ days

 

To view the work orders in any of the columns in the histogram the user clicks the relevant column of information; this will display the appropriate work orders in the working report. If a user report has been previously defined and saved by the user the work orders will be displayed in this format by default.

To configure the layout of the user interface and save that as a report for future use the user should use tools available in the Actions Menu.
