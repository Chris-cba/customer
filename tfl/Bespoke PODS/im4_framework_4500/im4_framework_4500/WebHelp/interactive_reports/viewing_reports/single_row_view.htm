Single Row View

To view the details of a single row at a time, click the single row view icon on the row you wish to view. If available, the single row view will always be the first column.

When selected, the record will be displayed. To return to the Report press the [Report View] button. The default Single Row View will display all Columns irrespective of whether they have previously been hidden using the Select Columns. Click the “Displayed Columns” check box to change the Single Row View to show only the Columns selected for Display. To exclude Columns with Null values click the “Exclude Null Values” checkbox.
