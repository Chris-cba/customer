Download

The Download option allows the current result set to be downloaded. The download formats will differ depending upon your installation and report definition but may include CSV, XLS, PDF, or RTF.

Control Breaks, Aggregates, Highlights and Calculated fields will not be included in CSV or XLS downloads.

Figure 64

When the appropriate download format has been selected a standard Windows “File Download” dialogue will be displayed allowing you to either open the report or save it to the required destination.
