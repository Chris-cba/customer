Select Columns

This option is used to modify the Columns displayed within the Interactive Report Pod. The columns on the right are displayed whilst the columns on the left are hidden. Columns may be selected from either side of the window and moved using the “Move Columns” toolbar.

Figure 42

You can reorder the displayed columns using the arrows on the far right of the window. Computed columns (see page 34) are prefixed with **.

Press [Apply] to invoke the column changes.
