Save Report

This option allows you to save your customized Report Pod for future use. Enter a name and optionally a Description for the customization.

When a Customized Report Pod is saved, it will be displayed as an additional Tab alongside the current “Working Report. To view a previously saved Report click on the required Tab. The example below shows the “Working Report” tab and 2 previously saved reports.

To Delete a previously saved customized Report Pod press the “Delete Report” icon for the report in question.
