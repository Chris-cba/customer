Map Info Tips

Map Info Tips are available by hovering the mouse cursor over an icon. Some info will be displayed about the object. The value displayed will be determined by the column defined as the “Label” column in the GIS Themes for the Layer. This is normally defined by your systems administrator.

“More Info” Callouts

More information can be viewed about you data by clicking on a legend icon. A call-out will be displayed showing you more details about the item.

This feature is currently not available for all layers.
