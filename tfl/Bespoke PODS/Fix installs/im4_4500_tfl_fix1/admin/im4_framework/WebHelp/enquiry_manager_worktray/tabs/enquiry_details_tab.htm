Category    Display Only

The type of the enquiry will be displayed.

Road     Edit

The road id will be displayed. This can be changed if required.

Description    Edit

The road description will be displayed. This can be changed if required.

Note

 

 

Once three characters of either the road or description have been entered into the appropriate field the system will automatically generate a list of candidate roads, as the number of characters are increased the candidate roads will be reduced to match the text entry.
