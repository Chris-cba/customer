Changing the order and naming of User Tabs

You can rename User Tabs with names that relate to their contents, re-order Tabs and switch off unused Tabs.

From the Config Tab select “Tabs”

Figure 82 Configuring Tabs

In the example in

Figure 82 all of the User Tabs are visible and have their default User 1..10 labels. The Admin Tab is accessible to the user (because they have been granted administrator rights of access) but the Tab has been set to not be visible.

The Home and Config Tabs cannot be removed or renamed through this interface.

Renaming User Tabs

Select the “edit” link on the tab you want to rename

The User Tab Name Dialog will be presented, change the default Tab name and add some descriptive details which will appear in the mouse over tooltip then select “Save”

Figure 83 Renaming a User Tab

Repeat this process for any other User Tabs that you wish to rename then save all of your changes by selecting the “Save” button.

Figure 84 Saving changes to Tab names, order and visibility

Once you have saved changes to User Tab Names the page will be refreshed and your new Tab names will be shown across the top of Information Manager

Setting the visibility of Tabs

You can set the visibility of any of the User Tabs (including the Business Areas, Query and Help Tabs). This feature is useful for hiding any unused User tabs.

The check box in each User Tab controls its visibility

Switch off any User Tabs that you do not want to be visible.
