Using the Calendar Date Picker

Figure 39

Several of the Information Manager Framework Tools and Pods use a Date Picker to allow you to select a date from a Calendar. When the Calendar is invoked it will default to the current system date as shown above. You can select the day and month required by scrolling through the months using the forward and back arrow and then clicking on the required day number, once an alternative to the current date is selected the popup will disappear.

Dates can alternatively be directly entered into search pods using the format 01-Jan-2011
