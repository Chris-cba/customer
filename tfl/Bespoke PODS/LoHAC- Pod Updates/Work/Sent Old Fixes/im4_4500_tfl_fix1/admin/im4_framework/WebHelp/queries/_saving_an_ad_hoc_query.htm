Saving an ad hoc query

If you wish to save a query for subsequent re-use click on and the following dialog will be presented. Enter a name and optionally a description of the query and then click on “Save” again.

Users will only have access to their own saved queries.

Figure 75 Save Query dialogue

The comparator values within your query will also be saved however these values can be changed when you return to the query. If you do not want to save any comparator values just leave them blank.
